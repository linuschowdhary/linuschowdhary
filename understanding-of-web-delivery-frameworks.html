Apache Sling

Apache Sling web framework=Apache Jackrabbit(JCR implementation)+Apache Felix(OSGi Framework and Service platform implementation)

So basically a framework that uses JCR implementation, Apache Jackrabbit in this case to store and manage content and Apache Felix, an OSGi implementation as dynamic runtime environment,dynamic in sense,code and content bundles can be loaded,unloaded and configured at runtime, without need to restart server for any configuration change.

In Sling,  JCR and OSGi is implemented in a way to assure that the apache sling is 

RESTful , content driven, supporting scripting languages like jsp,server side javascript ,scala etc.
